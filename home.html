HTML

< !DOCTYPE html > tells browser its HTML file
< head > Shows up in tab
< body > Shows up on page
< div > separates code into sections
< p > paragraph
< em > italics
< strong > bold
< br > new line
< ul > unordered list
< ol > ordered list
< li > list elements
< img src="link to file" alt="something"> image. alt will show if img doesn't load
< video src="link" width="" height="" controls> loads video. Text inbetween open and close will show if video doesn't load
< audio src="" controls> audio file
